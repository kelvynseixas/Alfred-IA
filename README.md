
# Alfred IA - Seu Mordomo Digital

Um sistema de gestão pessoal de luxo com foco em uma experiência de usuário fluida e interação por IA conversacional.

## Stack Tecnológica

- **Frontend**: React 19, TailwindCSS, Lucide Icons, Recharts
- **Backend**: Node.js, Express
- **Banco de Dados**: PostgreSQL
- **IA (Futuro)**: Google Gemini API

## Desenvolvimento Local

1.  **Configurar Backend:**
    -   Navegue até a pasta `server`: `cd server`
    -   Instale as dependências: `npm install`
    -   Crie um arquivo `.env` e adicione sua `DATABASE_URL` do PostgreSQL.
    -   Inicie o servidor: `npm start` (ou `node index.js`)

2.  **Configurar Frontend:**
    -   Na raiz do projeto, instale as dependências: `npm install`
    -   Inicie o servidor de desenvolvimento Vite: `npm run dev`

3.  **Acesso:**
    -   A aplicação estará disponível em `http://localhost:5173` (ou a porta que o Vite designar).
    -   O backend rodará em `http://localhost:3000`. O proxy do Vite já está configurado.
