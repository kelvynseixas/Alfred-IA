
[v1.0.6] - Módulo de Listas e Integração Financeira
- ADIÇÃO: Módulo de Listas (Suprimentos e Desejos) acessível via Sidebar.
- ADIÇÃO: Tabelas `lists` e `list_items` no Banco de Dados.
- ADIÇÃO: CRUD completo de Listas e Itens.
- RECURSO: "Ponte Financeira" - Ao concluir um item na lista, o sistema sugere lançá-lo imediatamente como despesa no financeiro, preenchendo automaticamente a descrição.
- MELHORIA: Refatoração do Dashboard para modularização.

[v1.0.5] - Painel de Tarefas (To-Do)
- ADIÇÃO: Módulo de Tarefas acessível via Sidebar.
- ADIÇÃO: Tabela de Tarefas (Tasks) no Banco de Dados (PostgreSQL).
- ADIÇÃO: Rotas de API para CRUD de tarefas e alternância de status (Concluído/Pendente).
- ADIÇÃO: Cards de KPI para tarefas (Total, Concluídas, Pendentes).
- ADIÇÃO: Lista de tarefas com priorização visual (Alta, Média, Baixa) e recorrência.

[v1.0.4] - Correções de Ambiente e Visualização
- CORREÇÃO: Arquivo .env do servidor forçado com as credenciais corretas fornecidas pelo usuário.
- CORREÇÃO: Lógica de filtro de data no Dashboard ajustada para evitar o problema de fuso horário onde lançamentos de "Hoje" apareciam como "Ontem" ou sumiam.
- CORREÇÃO: Tratamento reforçado de tipos numéricos no Backend para evitar erros de inserção no PostgreSQL.
- MELHORIA: Feedback visual (alert) mais detalhado em caso de falha ao salvar.

[v1.0.3] - Painel de Metas e Objetivos
- ADIÇÃO: Módulo de Metas acessível via Sidebar.
- ADIÇÃO: Tabela de Metas (Goals) e Histórico de Metas (Goal Entries) no Banco de Dados.

[v1.0.2] - Correção de Persistência e Módulo de Investimentos
- CORREÇÃO CRÍTICA: Implementada conexão real (API POST/PUT/DELETE) para movimentações financeiras. Antes os dados não persistiam.
- ADIÇÃO: Painel de Investimentos dedicado (acessível via Sidebar).
- ADIÇÃO: Tabela de Investimentos no Banco de Dados (PostgreSQL).
- ADIÇÃO: Formulário de Investimentos (Nome, Tipo, Rentabilidade %, Resgate).
- ADIÇÃO: Algoritmo de projeção de rentabilidade (1 mês, 6 meses, 12 meses).
- ADIÇÃO: Cards de KPIs de Investimento (Total Investido, Rentabilidade Média).
